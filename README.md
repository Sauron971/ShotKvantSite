# ShotKvantSite

This repository contains the website project created in the children's technopark **Quantorium**.

The project was developed to attract people to the VR/AR quantum direction and demonstrate the concept of an interactive experience connected with virtual and augmented reality technologies.

---

## About the project

The main idea of the project was to create a game that demonstrates the concept of VR/AR and helps users relax in their free time.

As planned, the game could be downloaded from any computer by accessing this website connected to the local network of the Quantorium and running.

However, the game itself was not fully completed.

---

## Repository contents

This repository contains:

* website source files;
* intermediate project materials;
* screenshots;
* photos used for project presentation.

---

## Purpose

The project was created as part of an educational activity to explore:

* web development;
* presentation of digital projects;
* VR/AR technology promotion;
* creating an interactive experience concept.

# ShotKvantSite

Данный репозиторий содержит сайт, созданный в детском технопарке **Кванториум**.

Проект был разработан для привлечения людей к направлению VR/AR и демонстрации идеи интерактивного проекта, связанного с технологиями виртуальной и дополненной реальности.

---

## О проекте

Основная идея проекта заключалась в создании игры, которая демонстрирует концепцию направления VR/AR и поможет расслабиться пользователям в свободное время.

По задумке, игру можно было бы скачать с любого компьютера зайдя на данный сайт, подключённого к локальной сети Кванториума, и запустить.

Однако сама игра не была полностью реализована.
